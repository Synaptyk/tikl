# Security policy

Keyboard remapping can affect passwords, login screens, remote sessions, accessibility tools, and system recovery.

Please report security-sensitive defects privately through the contact method published at tikl.my before opening a public issue.

The project does not currently claim that mappings are safe for firmware passwords, full-disk encryption prompts, operating-system login screens, remote desktop, kiosk environments, or assistive technology. Test with an emergency fallback.
